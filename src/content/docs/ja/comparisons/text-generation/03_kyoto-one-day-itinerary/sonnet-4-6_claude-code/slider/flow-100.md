夜明けと同時に動き出すのが京都の流儀です。**伏見稲荷大社**は朝6時前後が最も美しく、千本鳥居を独り占めに近い形で歩けます。夏の朝は日の出が早いため、5時半ごろ到着するのが理想的で、奥社奉拝所まで往復すると約90分。汗ばむ前に清涼な空気の中で歩き切れます。稲荷神社は24時間参拝可能で予約も不要ですが、早朝のJR稲荷駅は始発時刻の確認が必要です（京都駅から約5分）。

身体が目覚めたら、京阪電車で**祇園・東山エリア**へ移動します。9時の開門直後に**清水寺**へ向かうと、人波が押し寄せる前の静かな境内を歩けます。夏は緑が濃く、舞台からの眺望に霞が混じる独特の景観があります。入山料は500円で予約不要ですが、特別拝観（成就院庭園など）は事前にウェブサイトで開催スケジュールを確認してください。清水から三年坂・二年坂を下りながら散策し、10時を過ぎると店が開き始めるので、抹茶スイーツを楽しむのにちょうどよい時間帯になります。

そのまま祇園白川沿いを歩き、11時ごろ**建仁寺**へ。俵屋宗達の「風神雷神図屏風」（複製）と法堂の天井画「双龍図」は必見で、拝観料600円、予約不要。境内の竹林庭園は光の入り方が美しく、夏の強い日差しの中でも木陰が涼しいです。ここで午前中の行程を一区切りとして、**錦市場**へ移動しながら昼食をとるのが自然な流れです。錦は「京の台所」として旬の食材が並び、歩き食いしながら出汁巻き卵や湯葉、漬物を楽しめます。営業時間は店舗によって異なり、10〜18時頃の店が多いですが、午後2時前後から準備中になる店も出るため、昼食は早めに済ませるのがコツです。

午後は金閣・嵐山方向へ転じます。**嵐山の竹林（竹の径）** は夏でも涼しく、午後2〜3時台の光の角度が美しい写真を撮りやすいです。竹林自体は常時開放で無料ですが、隣接する**常寂光寺**や**野宮神社**なども組み合わせると時間の流れが豊かになります。この周辺でぜひ立ち寄りたいのが**天龍寺**の曹源池庭園で、世界遺産の中に富士山型の亀山を背景とした池泉庭園が広がっています（拝観料500円〜、要ウェブ確認）。嵐山は夏の日中は非常に混雑するため、16時以降に入ると人が引き始め、渡月橋からの夕景が楽しめます。

日が傾いてきたら、夕食の準備として予約確認が重要になります。京都の人気料亭や町家レストランは数週間前から満席になることが多く、**祇園や木屋町の割烹・懐石料理**は特に夏の時期（祇園祭シーズンの7月前後）は早めの予約が不可欠です。比較的予約が取りやすいのは**先斗町**のカウンター小料理屋や、**二条・御所周辺**の町家イタリアン・フレンチで、ランチタイムに電話して夕食を押さえる方法もあります。

夕食後は**鴨川の夕涼み**で締めくくります。先斗町から木屋町にかけての鴨川沿いでは夏限定で「床（ゆか）」が設けられ、川面を渡る風の中でビールや冷たい甘味を楽しめます。床の料亭は予約必須ですが、橋の上や河川敷に座るだけでも十分に京都の夏夜を体感できます。夜には**八坂神社**がライトアップされていることもあり、祇園祭の時期なら宵山の賑わいも加わって、京都らしい情緒が凝縮された時間になるはずです。
