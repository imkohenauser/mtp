## 方向性A：寺社と路地をゆっくり歩く「東山エリア」軸

**早朝（6:00〜8:00）**
- **清水寺**：夏の早朝は人が少なく、朝靄の中の舞台が格別。拝観は通常6:00〜。夏期は夜間特別拝観（ライトアップ）もある場合があるため、公式サイトで確認が必要。
- **二年坂・三年坂**：店が開く前の静かな石畳を歩く時間帯。

**午前（8:30〜11:00）**
- **高台寺**：豊臣秀吉の正室・ねねゆかりの寺。竹林も美しい。拝観時間：9:00〜17:30（要確認）。
- **建仁寺**：双龍図などの見どころ。9:00〜17:30、通常予約不要。

**昼（11:30〜13:30）**
- 祇園・花見小路エリアで昼食。**京料理・おばんざい**の定食を出す店も多いが、人気店は事前予約または整理券制の可能性あり。

**午後（14:00〜18:00）**
- **南禅寺〜哲学の道〜銀閣寺**ルート：水路閣・疏水沿いを歩く。銀閣寺は8:30〜17:00（夏期延長あり）。
- 途中の**法然院**（10:00〜16:00）などは知る人ぞ知る静かな名所。

**夕刻〜夜（18:00〜）**
- 鴨川の河原で夕涼み（納涼床は5〜9月、予約は1ヶ月前から埋まることも）。
- **八坂神社**は夜も参拝可能（境内は24時間）。

---

## 方向性B：嵐山〜嵯峨野エリアを早朝から制する

**早朝（5:30〜）**
- **竹林の道**：朝の光が差し込む時間帯が最も美しく、観光客もほぼゼロ。
- **天龍寺**：庭園拝観8:30〜（坐禅体験は予約制、公式サイトから）。
- **常寂光寺・二尊院**などの嵯峨野散策。

午後は移動して**伏見稲荷大社**（千本鳥居）へ。山頂まで2時間程度——夏は熱中症に注意が必要。

## 事前確認が必要な主なポイント

| 場所・体験 | 確認事項 |
|---|---|
| 夜間特別拝観（清水寺・高台寺など） | 期間・時間・料金は年によって異なる |
| 納涼床（川床料理） | 予約は1〜2ヶ月前が目安 |
| 人気料亭・寺院の精進料理 | 完全予約制が多い |
| 坐禅・写経体験（建仁寺・天龍寺など） | 事前申込が必要な場合がある |
| 祇園祭（7月）期間中 | 道路封鎖・大混雑が発生。宿泊も早めの確保が必要 |
