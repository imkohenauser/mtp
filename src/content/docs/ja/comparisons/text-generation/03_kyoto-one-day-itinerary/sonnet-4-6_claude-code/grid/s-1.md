# 夏の京都・特別な一日プラン

朝5時半。まだ誰もいない。**伏見稲荷大社**へ。千本鳥居を人影ゼロで歩ける唯一の時間帯がここにある——夏の早朝、空気がまだひんやりしていて、朱色がやばいほど鮮烈に見える。山頂まで歩いて90分、下りながら狐のお告げを妄想する時間も込みで。入場料ゼロ。ただし稲荷山登山は歩きやすい靴が必須。

8時。麓の茶屋で**稲荷茶屋の稲荷寿司**を食べる——ここでしか食べられない、甘めの揚げと酢飯のやつ。朝から食える。

10時に移動。**祇園祭の時期（7月1〜31日）なら山鉾巡行か宵山に命を賭けること。** これだけで一日終わる。鉾を間近で見ながら人波に飲まれる感覚、あれは別物。巡行は17日・24日、宵山は14〜16日・21〜23日。混雑は地獄だが後悔するなよ。

祇園祭でない日なら——午前中は**哲学の道〜南禅寺〜銀閣寺エリア**を歩き倒す。哲学の道沿いの小さな雑貨屋と甘味処を全部冷やかしながら。南禅寺の水路閣（赤レンガの水道橋）でカメラを構える外国人と並んで同じ写真を撮っても罪悪感なし。

12時。祇園・先斗町エリアへ移動して**昼食**。夏限定の**にしんそば**か**冷やし鱧料理**——京料理の夏の顔がここにある。先斗町は鴨川床（かわどこ）が出ているので川沿いに座れる店を狙う。**予約は絶対に前日までに入れること**。

14時〜16時。**二条城か京都御苑**で夏の暑さを受け入れながら歩く——いや、二条城の冷房の効いた大政奉還の間で歴史の重さに圧倒されながら涼む、でもいい。二条城は一般2時間見学が標準。開門9時、閉門17時（閉城16時）。**入場料一般1,300円。**

17時すぎ。**鴨川デルタ（出町柳）**で石飛びをする。鴨川と高野川が合流するあの場所、亀と市松模様の飛び石。子どもだろうが大人だろうが関係ない——跳ぶしかない。その後、**出町ふたばで豆大福**を買う。行列が常に存在するが17時前後は比較的短い。**売り切れ次第終了なので早めに動くこと。**

19時。日が落ちてもまだ暑い京都の夜へ。**高台寺か清水寺の夜間特別拝観**——夏のライトアップは別格。庭園が夜の光に沈む瞬間がある。高台寺は夏期拝観あり、料金600円、夜間21時30分最終受付が多い（**要公式サイトで確認！年によって変わる**）。

21時。締めは**祇園あたりの小さなバー**か、**鴨川沿いのコンビニ飯**——どっちも京都の正解。
