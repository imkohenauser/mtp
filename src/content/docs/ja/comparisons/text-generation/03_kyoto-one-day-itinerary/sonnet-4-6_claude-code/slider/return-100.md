## そもそも「一日観光プラン」は正しい問いか？

**前提を解体する**

「夏の京都での特別な一日」と聞くと、金閣寺・清水寺・嵐山を効率よく回るルートを思い浮かべる。しかしその前提を疑う価値がある。

夏の京都の現実は厳しい。気温は35〜38℃に達し、人気スポットは開門前から行列が形成される。「多くを見る＝特別」という等式は、炎天下の体力消耗と混雑の中で崩れやすい。問うべきは「何箇所回るか」ではなく、「夏の京都でしか成立しない体験は何か」だ。
