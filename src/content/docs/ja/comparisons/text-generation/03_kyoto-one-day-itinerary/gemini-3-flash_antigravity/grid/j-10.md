# 🌿 京の涼を愛でる、夏の特別観光プラン

コンセプトは**「水と緑に癒やされる、究極の避暑」**。市街地の喧騒と熱気を離れ、貴船の清流と伝統的な川床をメインに据えたプランです。

### 08:30 | 嵐山：朝の静寂と竹林の涼
- **場所：** 天龍寺 〜 竹林の小径
- **体験：** 観光客で溢れる前の、朝一番の嵐山を歩きます。竹林の中は街中よりも数度涼しく、笹の葉が擦れる音に癒やされます。
- **事前確認：** 天龍寺の参拝開始は8:30です。早朝の清々しい空気の中で「曹源池庭園」を眺めるのがおすすめです。

### 11:30 | 貴船：川床（かわどこ）で味わう京の極み
- **場所：** 貴船神社 〜 貴船川沿いの料理旅館
- **体験：** 市内中心部より5〜10度涼しい奥座敷・貴船へ。川の真上に設置された座敷「川床」で、せせらぎを足元に感じながら京会席を楽しみます。
- **★要予約：** **貴船の川床ランチは事前予約が必須**です（ひろ文、右源太など）。1ヶ月前には予約が埋まり始めるため、早めの手配を。予算は1万円〜2万円ほどです。

### 15:30 | 京都迎賓館：現代最高峰の職人技に触れる
- **場所：** 京都御苑内
- **体験：** 日本の伝統技能の粋を集めて建てられた、国賓を迎えるための施設です。洗練された建築と庭園は、夏の午後の強い日差しを忘れさせてくれる静謐な空間です。
- **★要確認：** **事前予約制（公式サイトより）**ですが、空きがあれば当日受付も可能です。ただし、国公賓の接遇などで急遽休館になることがあるため、必ず事前に公開カレンダーをチェックしてください。

### 19:00 | 鴨川：納涼床（のうりょうゆか）の夜景

- **体験：** 夕暮れの先斗町を散策し、鴨川沿いに並ぶ納涼床でディナーを。川風に吹かれながら、京都の夏の夜を締めくくります。
- **★要予約：** 鴨川沿いの床も大変人気です。和食だけでなく、イタリアンやフレンチの床もありますので、お好みのジャンルで2週間前には予約しましょう。

2.  **服装のドレスコード**
