**提案：洛北の水と青もみじの一日**

夏の京都で「特別感」を出すなら、街中の暑さと混雑を避けて、午前は鞍馬山、夕方から貴船の川床と貴船神社へ行く一日がよいです。山・水・神社・川床が一本の流れになり、京都らしさが濃い割に、祇園や清水周辺ほど消耗しません。

**モデル行程**

| 時間 | 内容 |
|---|---|
| 7:30頃 | 市内中心部を出発。京阪・叡電で出町柳から鞍馬へ |
| 9:00 | 鞍馬寺へ。仁王門から本殿へ、体力に応じて徒歩またはケーブル利用 |
| 10:30-12:30 | 鞍馬寺から奥の院経由で貴船へ山越え。暑い日は無理せず、鞍馬駅から叡電・バスで貴船口へ迂回 |
| 12:45 | 貴船神社を三社詣。本宮、結社、奥宮をゆっくり巡る |
| 14:00-16:00 | 貴船周辺で休憩。カフェ、川沿い散策、宿に戻って休む選択もあり |
| 17:00-18:00 | 貴船の川床で夕食。早めの時間帯が帰路も安心 |
| 日没後-20:00頃 | 7月上旬から8月中旬なら、貴船神社の七夕笹飾りライトアップを狙う |
| 20:00前後 | 京都バス・叡電で市内へ戻る。最終便は必ず当日確認 |

**この日の核**

朝は鞍馬寺。公式サイトでは本殿開扉が9:00-16:15、ケーブルは通常上り始発8:40・下り終発16:25と案内されています。愛山費は500円、ケーブル寄進は大人片道200円です。山道を歩くなら、サンダルではなく歩ける靴が前提です。  
出典：[鞍馬寺公式](https://www.kuramadera.or.jp/)、[入山に際して](https://www.kuramadera.or.jp/nyuzan.html)

昼から夕方は貴船。貴船神社は5月1日から11月30日の本宮参拝時間が6:00-20:00、年中無休・参拝料無料と案内されています。  
出典：[貴船神社公式](https://kifunejinja.jp/)

夜の特別感は川床です。右源太は京都市公式観光情報で、川床期間は5月-9月、営業時間は11:30-20:00、夜は17:00・17:30・18:00の3部制、雨天や増水時は屋内案内とされています。予約と、貴船口駅からの送迎は事前連絡が必要です。  
出典：[京都観光Navi 右源太](https://ja.kyoto.travel/tourism/single02.php?category_id=4&tourism_id=1317)

**事前確認が必要な点**

1. **川床の予約**  
   夏の貴船は人気が高いので、夕食は必ず予約してください。雨天・増水・台風時は川床ではなく屋内になる可能性があります。

2. **貴船神社ライトアップの日程**  
   2025年は7月1日-8月15日、日没から20:00頃まででした。2026年夏の正式日程は、訪問前に公式または京都観光Naviで確認してください。  
   出典：[京都観光Navi 七夕笹飾りライトアップ](https://ja.kyoto.travel/event/single.php?event_id=4549)、[京都府観光連盟](https://www.kyoto-kankou.or.jp/event/858)

3. **帰りの交通**  
   貴船は夜の便が限られます。京都バス「貴船」停留所の時刻表を当日確認し、川床の予約時間を逆算してください。  
   出典：[京都バス 貴船停留所](https://www.kyotobus.jp/route/timetable/kana07/kibune.html)

4. **山越えの可否**  
   雨の翌日、猛暑日、体力に不安がある日は、鞍馬寺から貴船への山道は避けてください。その場合は「鞍馬寺だけ参拝 → 叡電で貴船口 → バスで貴船」に切り替えるのが安全です。

5. **7月滞在なら祇園祭との組み合わせ**  
   7月なら、別日に祇園祭を入れる価値があります。祇園祭は7月1日-31日、山鉾巡行は前祭7月17日・後祭7月24日です。観覧席は販売制なので早めに確認してください。  
   出典：[京都観光Navi 祇園祭](https://ja.kyoto.travel/event/major/gion/)、[京都市観光協会 観覧席販売](https://www.kyokanko.or.jp/news/20260420_1/)

**一言でまとめると**

この一日は「朝は鞍馬の山で静けさ、夕方は貴船の水辺で涼、夜は灯り」を味わうプランです。真夏の京都では、名所を詰め込むより、涼しい北側に軸を置いたほうが記憶に残ります。

発展させるなら、滞在週が7月前半・祇園祭期間・8月16日前後のどれかで、夜の締め方を「貴船ライトアップ」「祇園祭宵山」「五山送り火」に差し替えると、さらに特別な一日になります。
