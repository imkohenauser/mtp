## 日本語の起源と歴史的な発展

### 起源

日本語の起源は言語学上の難問の一つです。現在のところ、孤立した言語系統（language isolate）とみなされており、明確な親族言語が確認されていません。

主な仮説として：
- **アルタイ語族説**：モンゴル語・トルコ語・満州語などとの共通祖語を想定するが、現在は否定的評価が主流
- **オーストロネシア語族との接触説**：南方からの渡来民による影響を示す語彙の類似
- **朝鮮語との関係**：文法構造の類似（SOV語順、助詞体系）が指摘されるが、系統的同一性は未証明

### 歴史的発展

**古代（〜奈良時代）**  
文字を持たなかった日本語は、5〜6世紀に漢字が伝来することで初めて書き記されるようになりました。『古事記』（712年）・『万葉集』（8世紀）では「万葉仮名」（漢字を音として借用）が使われています。

**平安時代（9〜12世紀）**  
漢字を簡略化した「仮名」（ひらがな・カタカナ）が発達。女性を中心に和語表現が花開き、『源氏物語』のような文学が生まれました。語彙・発音はこの時代と現代でかなり異なります（例：語末の「ゐ・ゑ・を」など消滅した音）。

**中世〜近世（13〜18世紀）**  
武家社会の台頭による武士ことばの普及、仏教語・禅語の流入、さらにポルトガル語からの借用（「パン」「タバコ」など）が起きました。江戸時代には共通語（江戸ことば）が形成されます。

**近代（明治以降）**  
西洋語の大量流入に対し、漢字を組み合わせた「和製漢語」（電話・哲学・社会など）が造語されました。1900年代の国語改革、そして1946年の「現代かなづかい」制定により、現代日本語の基盤が整います。

---

**一方、別の視点からも考えられます。**  
日本語の歴史は「輸入と吸収の歴史」として語られることが多いですが、逆に見れば「固有性を徹底的に保持してきた歴史」でもあります。中国語・英語という世界的大言語と接触しながらも、日本語は文法構造・音韻体系を根本的に変えることなく現在まで存続しています。語彙だけを借り、骨格を守り続けた——その保守性こそが、日本語の最も際立った特徴かもしれません。
