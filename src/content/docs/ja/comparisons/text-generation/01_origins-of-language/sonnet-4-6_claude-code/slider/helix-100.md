**Step 1: 問いの構造を確認**

「日本語の起源」と「歴史的な発展」は別問題。まず起源（どこから来たか）を考え、次に時代別の発展を整理する。

---

**Step 2: 起源の仮説を検討**

複数の説が存在する。どれが有力かを確認する。

- 仮説A「アルタイ語族説」 → 日本語の語順（SOV）・膠着語的形態がモンゴル語・朝鮮語と類似 → ただしアルタイ語族自体が現在は否定的。**棄却はできないが弱い**
- 仮説B「オーストロネシア語族との混合説」 → 語彙の一部に東南アジア・太平洋系の痕跡。縄文時代の言語との関係として提唱（大野晋） → 現在は主流ではない。**採用しない**
- 仮説C「孤立語説（語族不明）」 → 系統的親族が証明された言語は現時点でゼロ（琉球語は同じ日琉語族として確定） → **最も学術的に支持される立場**
- 仮説D「朝鮮語との関連」 → 構造的類似は大きいが、基礎語彙の対応が見つからない → **現時点では未証明**

**暫定結論①：日本語（および琉球語）は系統不明の孤立語族。起源地・経路は不確定。弥生時代（紀元前数世紀〜紀元後数世紀）に大陸から渡来した集団がもたらした言語と縄文系言語が融合した可能性が高い。**

---

**Step 3: 歴史的発展を時代別に整理**

仮説→確認→暫定結論の形で進める。

**3-1. 上代（奈良時代以前〜8世紀）**
- 仮説：最初期の日本語はどんな姿か？
- 確認：『古事記』（712年）・『万葉集』（8世紀末）に記録が残る。万葉仮名（漢字を音として借用）が最古の表記体系。母音が8つあった可能性（上代特殊仮名遣い）。
- 暫定結論②：この時点で日本語はすでに固有の語彙・文法を持ち、漢字を「借り物の文字」として使っていた。

**3-2. 平安時代（9〜12世紀）**
- 仮説：かな文字の発明で何が変わったか？
- 確認：9世紀に平仮名・片仮名が発生。漢文（男性貴族）とかな文学（女性・宮廷）が並立。母音は8→5に縮減。
- 棄却した視点：「女性がかなを使った」は社会的事実だが、言語変化の原因ではなく結果。
- 暫定結論③：表記体系の分岐が文体の分岐を生んだ。和漢混交文という新形式が生まれる素地もここに。

**3-3. 中世（12〜16世紀）**
- 仮説：武士社会の台頭が言語をどう変えたか？
- 確認：語頭ハ行の発音変化（pa→fa→wa）、文語と口語の乖離拡大、外来語（モンゴル語・禅の漢語）流入。
- 暫定結論④：権力中枢の移動（京都→鎌倉）が方言・語彙の多様化を促進。ただし書き言葉は古典的文語を維持。

**3-4. 近世（17〜19世紀）**
- 仮説：江戸時代に何が標準化されたか？
- 確認：江戸語が話し言葉の中心に。国学による古語研究。オランダ語・中国語からの語彙借用（蘭学）。
- 暫定結論⑤：印刷・出版の普及で「共通の書き言葉」への志向が生まれた。方言差は残りつつも、「江戸語」が後の標準語の基盤を形成。

**3-5. 近代（明治〜昭和初期）**
- 仮説：近代化で最大の変化は何か？
- 確認：言文一致運動（話し言葉≒書き言葉）、標準語政策（東京語を基準に全国教育）、欧米語からの大量借用（主にカタカナ語）。
- 棄却した視点：「明治に日本語が完成した」は誤り。この時期は意図的な再設計であり、現在もその影響が続いている。
- 暫定結論⑥：文語文法が制度的に廃止（戦後1946年の仮名遣い改定・当用漢字告示で確定）し、現代日本語が成立。

**Step 4: 私が迷った箇所の開示**

- 「起源」の扱い：言語学的系統（語族）の話か、日本列島での使用開始の話か、二つを混在させると混乱する。→ 両方に触れつつ区別することにした。
- 縄文語・弥生語の区別：証拠が極めて限られており、確定的なことは言えない。仮説レベルであることを明記した。
- 時代区分の粒度：「古代」「中世」「近代」では粗すぎるが、詳細すぎると主旨を見失う → 言語史上の転換点に絞った。

**結論**

日本語は**起源不明の孤立語族（日琉語族）**であり、縄文・弥生時代の言語接触を経て形成された可能性が高い。歴史的発展の要点は以下の5点：

| 時代 | 主要変化 |
|------|----------|
| 上代（〜8c） | 漢字借用による最初の文字記録、母音8つ体系 |
| 平安（9〜12c） | かな発明、文語/口語の分岐、母音5つへ縮減 |
| 中世（12〜16c） | 発音変化（ハ行）、文語と口語の乖離拡大 |
| 近世（17〜19c） | 江戸語の台頭、蘭学による外来語増加 |
| 近代（19c〜） | 言文一致・標準語政策・欧米借用語の急増、現代語の成立 |

現代日本語の「標準語」は明治〜昭和の意図的な制度設計の産物であり、それ以前の日本語とは表記・発音・語彙の面で大きく異なる。
